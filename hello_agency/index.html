Hello Agency

This VIZ module provides some general information about the agency linked with your VTFS API key. The information provided is contained in the agency's GTFS data.

The goal of this module is to provide an introductory example of the VTFS API and VIZ modules. Think about it as a "Hello World" script.

Now try it. Click the button and behold!

How does it work?

For agency users in the context of the VIZ console, various important configuration settings are automatically injected into your executed JavaScript in an object called userDetails, including your API key and your agency ID. These configuration details have been used to query the agency's VTFS API, and return information from the agency table of the GTFS schedule.

For developers working with the VTFS API outside of the VIZ console, the userDetails configuration object should be populated manually. Download this Hello Agency module on VIZX for an example of injecting the configuration, and use our API at https://vtfs.v-a.io/ for a list of available agencies. (Detailed documentation for our VTFS API is available at Apiary.)
